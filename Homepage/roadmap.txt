Тема проекта - Приложение для подсчета финансовых затрат
1. Сделать, чтобы при нажатии кнопки "Войти на сайт", появлялось
окошко с информацией "Вы ввели неправильный пароль или
такого пользователя не существует"
2. Создать базу данных с числовым и текстовым полем, куда будет
записана информация о затраченных деньгах
3. Добавить кнопку "+", при нажатии на которую открывается поле
ввода суммы трат
4. Добавить поле ввода текстового комментария для описания затраты
5. При нажатии на "ОК" выполнить миграцию в базу данных

6. На странице "Homepage" сделать ссылку "Показать историю затрат"
7. При переходе по ссылке будет выведена Дата, сумма и комментарий
8. Добавить графу Итого, который показывает общие затраты
